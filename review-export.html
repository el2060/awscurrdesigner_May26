2. Module Synopsis
✎ Edit ✦ Refine

This Level 2 module builds practical capability in core data structures and algorithms. Students learn to choose suitable approaches, implement and test solutions in Python, and reason about time/space trade-offs through labs and project work. The focus is on applying these concepts to realistic software problems.

7. Teaching & Learning Strategy
✎ Edit ✦ Refine

The module adopts a flipped-classroom approach combined with lab-based learning. Each week, students complete pre-class video tutorials and reading (OAL hours) covering theory and code walkthroughs. In-class sessions (IPL hours) are devoted to live problem-solving, code reviews, and pair-programming labs. Two summative projects build on weekly skills, with one mid-semester individual assignment and one team-based capstone in the final weeks.

StrategyHrs/weekMode
